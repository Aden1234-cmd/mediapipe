# 📱 MediaPipe 表情辨識 - 語音反饋應用

一個純網頁前端的實時表情識別應用，可以透過手機或電腦開啟相機，自動偵測使用者的表情並提供語音反饋。

## ✨ 功能特色

- **🎥 實時人臉偵測**：使用 MediaPipe Face Mesh 進行高精度人臉關鍵點偵測
- **😊 表情分類**：支援以下表情識別
  - 😊 開心（Happy）
  - 😢 傷心（Sad）
  - 😮 驚訝（Surprised）
  - 😠 憤怒（Angry）
  - 😐 中立（Neutral）
- **🔊 語音反饋**：根據偵測到的表情提供相應的語音回應
- **🌐 多語言支持**：繁體中文、簡體中文、英文
- **📊 即時表情顯示**：視覺化顯示各種表情的偵測分數
- **📱 响應式設計**：支援桌面、平板和手機等各種設備
- **⚙️ 可自定義設置**
  - 啟用/禁用語音反饋
  - 顯示/隱藏人臉關鍵點
  - 調整語音速度
  - 選擇語言

## 🚀 快速開始

### 方式一：直接開啟本地檔案

1. 在您的電腦上找到 `index.html` 檔案
2. 用瀏覽器雙擊打開或右鍵選擇「用瀏覽器打開」
3. 允許瀏覽器訪問您的相機
4. 點擊「🎥 開啟相機」按鈕開始表情識別

### 方式二：使用本地伺服器（推薦）

使用 Python：
```bash
# Python 3.x
python -m http.server 8000

# Python 2.x
python -m SimpleHTTPServer 8000
```

使用 Node.js：
```bash
npm install -g http-server
http-server
```

然後在瀏覽器中訪問 `http://localhost:8000`

## 📋 使用說明

### 基本操作

1. **開啟相機**：點擊「🎥 開啟相機」按鈕
2. **關閉相機**：點擊「⏹️ 關閉相機」按鈕
3. **清除記錄**：點擊「🗑️ 清除記錄」按鈕清空反饋日誌

### 設定選項

- **啟用語音反饋**：勾選此項以啟用語音回應功能
- **繪製人臉關鍵點**：勾選此項以在攝像頭中顯示人臉特徵點
- **語音速度**：調整語音合成的速度（0.5x ~ 2x）
- **語言**：選擇語音反饋的語言

### 表情偵測

應用會實時顯示以下資訊：
- 各種表情的偵測百分比
- 反饋記錄（最近的語音回應）
- 人臉關鍵點視覺化

## 🎯 表情識別原理

應用使用 MediaPipe 的 Face Mesh 模型來識別面部特徵，並根據以下特徵來判定表情：

| 表情 | 識別特徵 |
|------|--------|
| **開心** | 嘴角上升 + 眼睛有皺紋 |
| **傷心** | 嘴角下垂 + 眼睛稍微閉合 |
| **驚訝** | 眼睛大開 + 嘴巴張開 + 眉毛上升 |
| **憤怒** | 眉毛下沉 + 眼睛稍微閉合 |
| **中立** | 面部表情不明顯 |

## 🔧 技術架構

### 使用的技術
- **MediaPipe Face Mesh**：人臉偵測和關鍵點提取
- **Web Cameras API**：相機訪問
- **Web Speech API**：語音合成
- **Canvas API**：繪製人臉特徵
- **HTML5 + CSS3 + JavaScript**：前端介面

## 🔒 隱私和權限

- **相機權限**：應用需要訪問您的相機才能進行表情識別
  - 應用不會保存或上傳任何視頻或圖像
  - 表情識別完全在您的設備上進行

- **麥克風**：不需要麥克風（語音合成使用系統內置引擎）

## 🐛 常見問題

### 1. 無法訪問相機
- 檢查瀏覽器是否有相機權限
- 確保沒有其他應用佔用相機
- 嘗試重新啟動瀏覽器

### 2. 表情識別不準確
- 確保光線充足
- 保持臉部在相機框內
- 調整距離（保持 30-60 公分）

### 3. 語音反饋沒有聲音
- 檢查電腦/手機的音量設定
- 確保系統語言設定正確
- 嘗試選擇不同的語言

### 4. MediaPipe 加載失敗
- 檢查網路連接
- 清除瀏覽器快取
- 嘗試使用不同的瀏覽器
